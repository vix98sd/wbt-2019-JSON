JSON struktura

JSON predstavlja tekstualnu datoteku čiji format podseća na format objekta koji podržava programski jezik JavaScript.

JSON takođe podržava iste tipove podataka kao i standardni JavaScript objekti:

    • Prosti tipovi

    • String (kao tekstualni tip)

    • Number (kao brojčani tip)

    • Boolean (kao istinitosni tip)

    • null (kao nepostojeća vrednost)

    • Složeni tipovi

    • Array (kao niz prostih tipova)

    • Ostali objektni literali
