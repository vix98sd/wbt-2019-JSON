JSON struktura

JSON predstavlja tekstualnu datoteku čiji format podseća na format objekta koji podržava programski jezik JavaScript. Iako je kreiran po ugledu na JavaScript, može se koristiti i van njega. JSON je format podataka, što znači da ne može obuhvatati metode. I najsitnija greška pri korišćenju zareza može izazvati JSON grešku i učiniti da datoteka ne radi.

Objekti čine uređen par naziv - vrednost. Objekti se okružuju vitičastim zagradama ("{" i "}"). Inicijalni objekat se ne imenuje, dok se interni imenuju korišćenjem tekstualnog tipa podatka i znakom ":".

Nizovi predstavljaju listu vrednosti. Kao što je već navedeno, JSON ima izgled sličan JavaScript objektu. Sličan, iz razloga što je niz kao takav samodovoljan da predstavlja JSON objekat. Nizovi se okružuju uglastim zagradama ("[" i "]"). Pristup elementima niza vrši se preko indeksa, koji, kao i kod programskog jezika JavaScript, počinju od 0.

    • null (kao nepostojeća vrednost)

    • Array (kao niz prostih tipova)

Objašnjenje primera

Sav sadržaj JSON datoteke sadržan je u inicijalnom objektu, ograničenom vitičastim zagradama i neimenovanom. Interni objekti se mogu prepoznati po izgledu koji karakterišu naziv, dvotačka i vrednost. Na ovom primeru ističu se:
"Naziv": "Ubiti pticu rugalicu",
"Author": "Robert Galbraith",
"Žanr": "klasik krimi novela",
i ostali.

Labela "Detalji" sadrži u sebi objekat, koji se, ponovo, sastoji od nekoliko parova naziv - vrednost.

Labela "Cena" sadrži naziv, koji sadi dva različita objekta pod istom jurisdikcijom. Ova labela može biti i primer ugnježdavanja (slučaj kada tip predstavlja element drugog tipa) u JSON-u.
Važno! Brojevi nikada nisu okruženi znacima navoda!

Pristup elementima

Za pristup elementima koristi se hijerarhijski pristup (od roditelja ka detetu) i notacija tačka/uglasta zagrada, karakteristična za JavaScript, i to:

Za pristup broju strana koriste se imena svojstava koja vode do atributa razdvojena tačkama ili ograđena uglastim zagradama.
Detalji.Strana
ili
["Detalji"]["Strana"]

Za pristup ceni džepnog izdanja koriste se imena svojstava koja vode do atributa i indeksi niza kom cena pripada, razdvojena tačkama ili ograđena uglastim zagradama.
Cena.[1]
ili
["Cena"][1]
