Više o JSON-u

JSON (JavaScript Object Notation) je jedan od lakših tekstualnih otvorenih standarda dizajniran za čitljivu razmenu podataka. Ekstenzija datoteke s podacima u JSON-ovom formatu je .json, dok je meta oznaka (MIME format) application/json.

JSON je format koji polako zamenjuje XML jer ima nekolicinu prednosti u odnosu na XML.

Kada je oblast primene u pitanju, JSON, kao i XML standard se najčešće mogu prinaći u komunikaciji između web servera i klijenta. Konkretno, koriste se za primanje podataka poslatih od strane web servera.

JSON ne koristi tagove pa stoga ima kraći kod koji je lakši za pisanje i razumljiviji za čitanje.
